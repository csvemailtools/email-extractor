Frequently Asked Questions

Is this tool really free?
Yes, completely free. There are no limits on file size, number of rows, or number of downloads. The tool is supported by ads.
Are my files uploaded to a server?
No. All processing happens directly in your web browser. Your files are never sent anywhere — they stay on your device the entire time.
What formats are supported?
CSV (.csv), Excel 2007+ (.xlsx), and older Excel (.xls) files are all supported. The tool reads the first sheet of Excel files.
How does email validation work?
The tool checks each value against a standard email format pattern — it must have a local part, an @ symbol, a domain, and a valid TLD. It does not send test emails or ping mail servers.
What does "split into N files" mean?
If you have 10,000 emails and split into 4 files, you get four CSVs of about 2,500 rows each. This is useful for email marketing platforms that have per-campaign limits.
Can I extract any column, not just emails?
Yes. You can select any column from your file. The email validation option only applies when you specifically need it — you can leave it off for phone numbers, names, or any other data type.
